# 私有配置模块

公开仓库只包含程序源码，不包含稿件、作者、期刊映射、风险报告、API Key 或网页登录凭据。

## 在原电脑导出

双击 `导出私有配置模块.bat`。工具会在 `.local/exports/` 生成一个 `.kcbundle` 文件，其中只包含连接与转换配置，不包含稿件数据。

导出范围包括：

- 工作台 AI 配置
- 稿件 Excel 转 SQLite 所需的映射 CSV
- 伦理审查与引用检查的本机配置
- Google Drive/出版社数据同步所使用的本地凭据文件（如果存在）
- `.local/publisher_accounts.json`（如果后续建立该文件）

`.kcbundle` 不包含 Excel、SQLite、稿件、作者或风险报告，但仍可能包含账号和 OAuth 令牌。请仅通过可信介质传输，不要发送到公开网盘或提交到 GitHub。

## 在新电脑导入

1. 从 GitHub 下载或克隆公开项目。
2. 安装 `requirements.txt` 中的依赖。
3. 双击 `导入私有配置模块.bat` 并选择 `.kcbundle` 文件，也可以把该文件拖到批处理文件上。
4. 双击 `同步并重建稿件数据.bat`，从 Google Drive 下载最新 Excel 并转换成 SQLite。
5. 重新启动工作台。

导入前会校验文件清单和 SHA-256；覆盖已有文件时会先备份到 `.local/import-backups/`。

## 命令行

```powershell
python tools/private_module.py export
python tools/private_module.py inspect "D:\path\editorial-private-config.kcbundle"
python tools/private_module.py import "D:\path\editorial-private-config.kcbundle"
```
